UI : User Interface
API : Application Programming Interface

사용자가 시스템을 제어하기 위해서 사용하는 조작 장치를 UI라고 부른다.

웹 페이지에서 뜨는 경고 창의 경우 우리의 의도가 반영되었으나, 기능이나 모양 등의 내용이 
들어가 있지 않다. 해당 경고창은 웹브라우저의 개발자들이 우리 대신 만들어 두었다가 특정 함수를
실행하면 경고창을 띄워주겠다고 약속한 것이다.
함수 alert는 경고창을 실행하는 조작장치인 것이다.
애플리케이션을 만들기 위해서 프로그래밍을 할 때 사용하는 조작장치들을 API라고 한다.

자신의 프로젝트를 실행할 떄
모든 개념을 총동원하려고 하지 말아라.
최소한의 도구들만 가지고 해결하려고 하라.
순서대로 실행된다는 것 만으로도 충분히 혁명적인 것이다.
최소한의 도구 -> 다른 도구들 사용 시작 -> 각각의 개념들간에 익숙해지는 과정 -> 한계 -> 다시 공부
